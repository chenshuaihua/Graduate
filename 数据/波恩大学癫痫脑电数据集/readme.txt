德国波恩大学癫痫脑电数据集

波恩大学癫痫中心数据库是在 2001 年建立，并逐渐成为脑电信号分类研究中的常用标准数据库。该数据库包含 5
组脑电信号( A ～ E),通过173.61Hz频率采样得到，每组有100段脑电信号，每段信号包含4097个数据点
单位应该是uv
发作间期：病患无明显发病症状
发作前期：癫痫发作前的数秒到数分钟，在这个阶段，病患可能会表现出错觉、幻觉、局部肌阵挛等前驱症状
发作期：病患的全身性大发作、失神性发作和部分性发作阶段。
癫痫脑电中的异常波形多集中在0.5-35Hz之间。在临床上，将这些异
常波形称之为癫痫样放电，主要有尖波、棘波、尖慢复合波、棘慢复合波等几种形式。


A_Z组（健康志愿者睁眼时的正常脑电信号）
B_O组（健康志愿者闭眼时的正常脑电信号）
C_N组（癫痫患者发作间期病灶外的脑电信号）
D_F组（癫痫患者发作间期病灶区的脑电信号）
E_S组（癫痫患者发作期的脑电信号）


1024个数据点算是一组
